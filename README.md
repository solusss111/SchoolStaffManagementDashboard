# 🏫 School Staff Management Dashboard

A web dashboard for managing school staff, categories, and attestation periods.

## 📌 Description

This project is a React + TypeScript dashboard application for managing school employees.

Features:
- View staff list
- Filter by category, subject, and attestation year
- Import staff data from Excel
- Status indicators for attestation
- Modern responsive UI

This project was created as a portfolio project.

---

## 🛠 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- xlsx (for Excel import)

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/school-staff-management-dashboard.git
```
Install dependencies
```bash
npm install
```
Run development server
```bash
npm run dev
```
